# Entire Scoop Bucket

Private Scoop bucket for the Entire CLI.

## Installation

First, add the bucket:

```powershell
scoop bucket add entire https://github.com/entireio/scoop-bucket.git
```

Then install:

```powershell
scoop install entire/cli
```

## Updating

```powershell
scoop update
scoop update cli
```

## Uninstall

```powershell
scoop uninstall cli
scoop bucket rm entire
```
